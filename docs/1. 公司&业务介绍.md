## 一、公司基本情况

### 1.1 公司概况
+ **全称**: 维他很忙零食有限公司
+ **英文名**: Vitamin Retail Co., Ltd.
+ **成立时间**: 2015年
+ **总部地址**: 中国上海
+ **公司性质**: 全国连锁零食新零售企业
+ **业务模式**: 线下直营+加盟、电商、批发多业态

### 1.2 企业规模
+ **门店规模**: 约1000家门店（直营600家+加盟400家）
+ **覆盖城市**: 全国150+城市
+ **商品规模**: 5000+ SKU
+ **仓储网络**: 全国8-10个区域仓/中心仓+50+前置仓
+ **员工规模**: 
  - 总部员工: 500-800人
  - 门店员工: 5000-8000人
+ **年营业额**: 约50-80亿元人民币

## 二、业务板块

### 2.1 零售业务（主业务 - 占比70%）

#### 线下门店即时零售
+ **门店类型**:
  - 标准店(80-120㎡): 社区商圈、写字楼
  - 旗舰店(150-200㎡): 核心商圈、购物中心
  - 迷你店(30-50㎡): 地铁站、学校周边
+ **经营方式**:
  - 自助选购+收银台结算
  - 散称商品+预包装商品
+ **营业时间**: 7:00-23:00

#### 门店O2O业务
+ **接入平台**: 美团外卖、饿了么、自有小程序
+ **配送模式**: 
  - 平台骑手配送
  - 门店自配(3公里内)
+ **配送时效**: 30分钟达

### 2.2 电商业务（次要但增长快 - 占比20%）

#### 平台电商
+ **入驻平台**: 天猫、京东、拼多多、抖音、小红书
+ **运营模式**: 
  - 旗舰店官方直营
  - 分销商授权经销

#### 自营电商
+ **自有渠道**: 官方APP、微信小程序、官网商城
+ **履约模式**:
  - 仓配一体(区域仓发货)
  - 门店发货(就近门店)

### 2.3 批发与分销业务（补充业务 - 占比10%）

#### B2B批发
+ **客户类型**:
  - 传统经销商
  - 便利店/夫妻店
  - 团购客户(企业福利)
+ **结算模式**:
  - 预付款
  - 月结账期(30-60天)

#### 代销/联营
+ **合作模式**:
  - 供应商在门店设专区
  - 按实际销售结算
  - 库存归供应商

## 三、商品体系

### 3.1 商品分类

#### 核心品类
+ **休闲零食**: 坚果炒货、膨化食品、肉干肉脯、糖果巧克力
+ **饮品**: 功能饮料、果汁茶饮、咖啡奶茶
+ **冷冻冷藏**: 冰淇淋、冷冻甜品、冷藏鲜食
+ **进口零食**: 日韩、欧美、东南亚进口商品

#### 商品特性
+ **SKU数量**: 5000+
+ **新品引入**: 每月50-100个新品
+ **商品淘汰**: 每季度清退200-300个滞销品
+ **保质期管理**:
  - 常温商品: 3-12个月
  - 冷藏商品: 7-30天
  - 冷冻商品: 3-6个月

### 3.2 多温区管理
+ **常温区**: 占比60%
+ **冷藏区(2-8℃)**: 占比25%
+ **冷冻区(-18℃以下)**: 占比15%

## 四、供应链体系

### 4.1 采购模式
+ **集中采购**: 总部统一采购全国商品
+ **区域采购**: 区域特色商品由区域公司采购
+ **门店直采**: 少量生鲜、地方特产

### 4.2 仓储网络

#### 中心仓
+ **数量**: 8个
+ **覆盖**: 华东、华南、华北、西南、华中等
+ **功能**: 大批量采购存储、跨区域调拨

#### 区域仓
+ **数量**: 30+个
+ **覆盖**: 省级/大区级
+ **功能**: 门店补货、电商发货

#### 前置仓
+ **数量**: 50+个
+ **覆盖**: 重点城市
+ **功能**: O2O快速履约

### 4.3 配送网络

#### 干线物流
+ **模式**: 自建车队+第三方承运
+ **路线**: 中心仓↔区域仓
+ **频次**: 每周2-3次

#### 城配物流
+ **模式**: 第三方城配公司
+ **路线**: 区域仓→门店
+ **频次**: 每周3-5次

#### 冷链物流
+ **合作方**: 顺丰冷运、京东冷链
+ **温控**: 全程温度监控
+ **时效**: 24-48小时

## 七、行业特性

### 7.1 零食新零售行业特点
+ **高频消费**: 顾客复购周期1-2周
+ **短保质期**: 大量商品保质期短，临期管理压力大
+ **多温区**: 常温/冷藏/冷冻三类场景
+ **高频补货**: 门店库存有限，每周补货3-5次
+ **促销密集**: 每日促销+会员活动+节庆大促

### 7.2 业务挑战
+ **库存周转**: 控制临期/过期商品损耗
+ **供应链效率**: 高频补货对物流时效要求高
+ **多渠道协同**: 线上线下库存共享、价格统一
+ **食品安全**: 批次追溯、质量管控要求严格
+ **成本控制**: 租金、人工、物流成本持续上涨

## 八、技术与系统现状

### 8.1 现有系统
+ **门店POS系统**: 收银、店务管理
+ **电商OMS**: 订单管理、库存管理
+ **WMS**: 仓库作业管理
+ **CRM**: 会员管理、营销活动
+ **旧版ERP**: 采购、库存、财务（急需重构）

### 8.2 系统痛点
+ **系统孤岛**: 各系统独立建设，数据不互通
+ **数据不一致**: 库存、会员、订单数据常出现差异
+ **扩展性差**: 难以支撑业务快速增长
+ **手工作业多**: 大量Excel表格、线下流程
+ **决策滞后**: 缺少实时数据分析能力
